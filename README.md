# HireX – AI-Powered Recruitment & Resume Screening Platform

## Overview
HireX is an AI-powered hiring platform that helps recruiters screen resumes and match candidates with suitable job opportunities efficiently.

## Features
- Resume Upload & Analysis
- AI-Based Candidate Screening
- Job Matching System
- Recruiter Dashboard
- Application Tracking

## Tech Stack
- HTML
- CSS
- JavaScript
- Python
- Flask
- Supabase
- Vercel

## Project Goal
To simplify recruitment by automating resume screening and candidate-job matching.
